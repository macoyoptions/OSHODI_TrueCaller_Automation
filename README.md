MY STEP-BY-STEP GUIDE ON HOW TO I CREATED THIS FRAMEWORK USING APPIUM, PYTHON AND PYTEST TO RUN THIS LOGIN SCRIPT
Firstly, i created a GitHub REPO
I clone the REPO to my local system
Use this command in your terminal or command prompt:git clone and add the repo URL and click enter
I use my selected IDE which is pycharm to open the cloned Repo
I created requirement.txt using this command pip install -r requirements.txt
I install Appium and other required libraries: Using pip install Appium-Python-Client
install selenium, command pip install selenium
Install webdriver-manager, command pip Install webdriver-manager
Install pytest, command Install pytest
Python Package was created from the OSHODI_TrueCaller_Automation and call it LoginPageTest, ON the LoginPageTest package created a python file LoginPage
